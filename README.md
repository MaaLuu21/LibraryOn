# LibraryOn
LibraryOn é um projeto que simula uma biblioteca online. Nele, serão documentadas todas as etapas do desenvolvimento: criação dos casos de uso, organização da aplicação em camadas, construção da API, comunicação entre os módulos e configuração da infraestrutura. Também existe a possibilidade de desenvolver um front-end simples para interação com o usuário.
